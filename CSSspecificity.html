CSS Specificity

CSS Specificity?!

CSS Sepecificity เป็นค่าบางค่าที่จะบอกให้ browser ไปเลือก selector จากมากไปน้อย ถ้ามีค่ามากก็จะถูกเลือกไปใช้ และ CSS ก็เหมือนภาษาการเขียนโปรแกรมอื่น ๆ ที่อ่านโค้ดจาก บน – ล่าง ดังนั้นถ้าเราเขียน Selector ซ้ำกัน อันล่างจะถูกนำไปใช้ เช่น

.test { background-color: blue; }

.test { background-color: green; }

.test { background-color: red; }

    วิธีคำนวณค่า CSS Specificity

  • 1 แบบ Inline style มีค่า specificity = 1000
  • 2 แบบ ID selectors มีค่า specificity = 100
  • 3 แบบ Class selectors, Attributes selectors, Pseudo-classes มีค่า specificity = 10
  • 4 แบบ Type selectors, Pseudo-elements มีค่า specificity = 1
  • 5 แบบ Universal selector, Combinators, Negation pseudo-clas มีค่า specificity = 0

    ตัวอย่างการคำนวณ

  • a { } = 1 คะแนน
  • .someClass { } = 10 คะแนน
  • #someId { } = 100 คะแนน
  • ถ้าเขียน css ใน style เลย = 1000 คะแนน
  • div a { } = 2 คะแนน (div = 1) + (a = 1)
  • .test a { } = 11 คะแนน (.test = 10) + (a = 1)
  • .test a #id { } = 111 คะแนน (.test = 10) + (a = 1) + (#id = 100)
